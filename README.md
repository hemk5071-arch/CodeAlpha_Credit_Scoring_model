# 💳 Credit Scoring Prediction using Machine Learning

## 📌 Project Overview

The **Credit Scoring Prediction** project is a machine learning application that predicts whether a customer is likely to have **low credit risk** or **high credit risk** based on their financial and credit history.

The model analyzes multiple credit-related factors such as payment history, credit utilization, inquiries, and trade line information to assist financial institutions in making informed lending decisions.

---

## 🚀 Features

* Predicts customer credit risk.
* Machine Learning-based classification model.
* Uses Logistic Regression for prediction.
* Model saved using Python Pickle (`.pkl`).
* Supports prediction for new customer data.
* Clean and reusable project structure.

---

## 🛠️ Technologies Used

---

## 📂 Dataset Features

The model is trained using customer credit information including:

* Derogatory Count
* Collection Count
* Bankruptcy Indicator
* Inquiry Count (Last 6 Months)
* Finance Inquiry Count
* Trade Line History
* Total Trade Lines
* Total Loan Amount
* Maximum Loan Amount
* Satisfied Trade Lines
* Delinquency Counts
* Credit Utilization
* Open Account Percentage
* Credit Balance Percentage
* And other credit-related financial attributes.

---

## ⚙️ Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training using Logistic Regression
6. Model Evaluation
7. Model Serialization using Pickle
8. Prediction on New Customer Data

---

## 📊 Model

* Algorithm: Logistic Regression
* Task: Binary Classification
* Output:

  * Low Credit Risk
  * High Credit Risk
